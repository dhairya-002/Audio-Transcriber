# Audio Transcribe Application

A full-stack audio transcription application built with React and Spring Boot, using OpenAI APIs.

## Tech Stack
### Frontend
- React
- Vite

### Backend
- Java
- Spring Boot
- Maven

## Features
- Upload audio
- Transcribe audio using AI
- REST API backend

## Run Locally

### Backend
```bash
cd audio-transcribe-backend
mvn spring-boot:run
